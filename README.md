# 📝 React To-Do List App

A simple and clean Note application built with **React**. You can add, delete, and manage your Notes easily.

## 🚀 Features

- Add new note
- Delete notes
- Responsive layout
- Clean UI using CSS

## 🛠️ Technologies Used

- React
- JavaScript
- HTML & CSS 
- Local Storage (optional)
